Assembly Language

In this class we learned how to create programs using Risc-V. We learned to coded the Tower of Hanoi game and a number conversion from binary, Decimal, and Hexadecimal. It is a really hard language for those who are used to the way that you write with high-level languages.It gives a lot of control, but it needs a line of code for every single movement it makes. One of the last assignments we made in the class was an atari game. It was a group project. My group decided to go with a combination of snake and frogger. While it was never finsihed, we did get it to work to a certain extent. Firstly, i created the sprite. It was an accident, but it became a frog looking creature. The frog had to be able to move across the screen and collect little balls. The balls would then spawn in a different spot once it interacted with the sprite. It took a lot of effort and late nights, but we eventually got the frog to be able to interact with the ball and move around on screen.

The link for the assignments for this class can be found at this GitHub Repository.
