Assembly Language

In this class we learned how to create programs using Risc-V. We learned to code the Tower of Hanoi game and a number conversion from binary, Decimal, and Hexadecimal. It is a really hard language for those who are used to the way that you write with high-level languages. It gives a lot of control, but it needs a line of code for every single movement it makes. One of the last assignments we made in the class was an atari game. It was a group project. My group decided to go with a combination of Snake and Frogger. While it was never finished, we did get it to work to a certain extent. Firstly, I created the sprite. It was an accident, but it became a frog-looking creature. The frog had to be able to move across the screen and collect little balls. The balls would then spawn in a different spot once they interacted with the sprite. It took a lot of effort and late nights, but we eventually got the frog to be able to interact with the ball and move around on screen.

The link for the assignments for this class can be found at this GitHub Repository.

I am working my way through a Risc-V book on the language again, because while I didn't understand it then, I liked the language and simplicity through which everything worked. It was hard to learn as there weren't many resources to be used. The resource we used the most was the online CREATOR compiler from GitHub.

Lizard Adoption

As another class project, the Adopt a Pet page is a simplistic page designed to convince my teacher to get a frilled-neck lizard as his next pet. It gives a species overview, care instructions, behavior, and whether or not it was legal to own in the state of Arkansas. It includes text, hyperlinks, pictures, and a table. It is easy to see the improvement that I have made in this project.

Blog

This project was part of a class assignment focused on learning HTML, CSS, and JavaScript. The end goal was to create a website following the Dos and Don'ts my professor created. I created a photography portfolio and blog.
It included a home page, an image gallery allowing you to move through the different topics and pictures available, an information page telling the tools I used, and a blog. It is currently unfinished, but if I were to go back and finish it, it would have a working internal navigation for the blog page, better structure, and a lightbox functionality for the pictures.
